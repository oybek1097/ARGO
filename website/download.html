Download & install

Pick the option that fits your environment. The argo-brain core runs on the Python standard library alone — so you can try ARGO with nothing more than a Git clone.

Install options

All options are self-hosted. ARGO never requires a hosted account to run.

Try it — no install

Fastest

Clone the repository and start chatting. argo-brain uses only the Python 3 standard library, so there are no dependencies to install. 

git clone https://github.com/argo-agent/argo.git
cd argo/argo-brain
python3 -m argo_brain chat      # interactive conversation, no API key
python3 -m argo_brain doctor    # environment diagnostics

One-shot setup script

Recommended

The bundled installer checks your toolchain, builds the argo-core Rust gateway and runs the configuration wizard. 

git clone https://github.com/argo-agent/argo.git
cd argo
./scripts/setup.sh

Docker

Alpha

Run ARGO as a container — convenient for evaluation and single-node deployments. 

# build the image from the repository
docker build -t argo-agent:latest .

# run the gateway
docker run -p 8080:8080 argo-agent:latest

Kubernetes (Helm)

Alpha

Deploy ARGO to a cluster with the Helm chart — suitable for sovereign, self-managed installations. 

helm install argo ./deploy/helm/argo \
  --namespace argo --create-namespace

PyPI package

Planned for GA

Once published, the brain will be installable directly from PyPI. Until then, install from a local checkout of the repository. 

# planned for the GA release
pip install argo-brain

# available today — install from source
pip install ./argo-brain

Native build

Best performance

Build the argo-core Rust gateway from source for the smallest footprint and fastest cold start. 

cd argo-core
cargo build --release
./target/release/argo-core
Heads up: ARGO v3.0 is alpha software approaching general availability. It is suitable for evaluation, development and pilot deployments — please do not treat it as production-ready yet. Container images, the Helm chart and the PyPI package are firming up ahead of the GA launch. See the documentation for full instructions.

System requirements

ARGO is deliberately lightweight and runs comfortably on modest hardware.

argo-brain

Python 3.9 or newer. Standard library only — no third-party packages required to run the core.

argo-core

A recent Rust toolchain to build from source. The resulting binary is roughly 1.3 MB and idles under 20 MB of RAM.

Platforms

Linux and macOS. Self-hosted on your own hardware, in your own cloud, or fully airgapped.
