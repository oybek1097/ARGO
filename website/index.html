Why ARGO

Sovereign deployment

ARGO is self-hosted first. Run it entirely inside your own network — including fully airgapped, with no external internet access. Layered memory and compliance-oriented modules keep data residency under your control, which matters for governments, banks and privacy-first teams.

  • Self-hosted, MIT licensed — no usage metering, no lock-in
  • Airgapped mode for regulated environments
  • Bring your own LLM provider, or run a local model

Central Asia focus

Most agent platforms treat Central Asian languages as an afterthought. ARGO makes them first-class: tokenization, language detection and response routing for Uzbek, Russian, Kazakh, Kyrgyz and Tajik are part of the core — serving a region of 250M+ people that is currently underserved.

  • Native language detection across uz / ru / kk / ky / tg / en
  • Localized response routing out of the box
  • Built for CIS DevOps teams and regional startups

Try it in under a minute

The argo-brain core runs on the Python standard library alone — clone the repo and start chatting, no dependencies to install.

bash 
# clone the repository
git clone https://github.com/argo-agent/argo.git
cd argo

# one-shot install (checks toolchain, builds argo-core, configures)
./scripts/setup.sh

# or just try the brain — no install required
cd argo-brain
python3 -m argo_brain chat      # interactive conversation, no API key
python3 -m argo_brain doctor    # environment diagnostics
python3 -m argo_brain serve     # start the HTTP gateway

Prefer Docker, Helm or PyPI? See the download & install page.
