# Урок 01 — Установка ARGO за 5 минут (Russian)

**Тема:** Установка
**Язык:** Русский
**Целевая длительность:** ~4 минуты

---

### Сцена 1 — Вступление (0:00–0:25)

ON SCREEN: логотип ARGO, затем чистое окно терминала.

NARRATION: «Здравствуйте и добро пожаловать в ARGO. ARGO — это
платформа ИИ-агента с открытым исходным кодом, которую вы запускаете
у себя. За несколько минут мы установим её с нуля. Хорошая новость:
устанавливать почти нечего — мозг ARGO работает только на стандартной
библиотеке Python.»

### Сцена 2 — Что понадобится (0:25–0:55)

ON SCREEN:
```
python3 --version
git --version
```

NARRATION: «Вам нужны две вещи: Python 3 и Git. Этого достаточно, чтобы
попробовать ARGO. Если позже вы захотите собрать Rust-шлюз, понадобится
ещё инструментарий Rust, но для этого видео он не нужен.»

### Сцена 3 — Клонируем репозиторий (0:55–1:35)

ON SCREEN:
```
git clone https://github.com/argo-agent/argo.git
cd argo
```

NARRATION: «Сначала клонируйте репозиторий и перейдите в его папку. ARGO
состоит из двух частей: argo-core — это Rust-шлюз, и argo-brain — это
Python-мозг. Сейчас мы будем работать с мозгом.»

### Сцена 4 — Запускаем скрипт установки (1:35–2:40)

ON SCREEN:
```
./scripts/setup.sh
```

NARRATION: «В ARGO есть скрипт установки, выполняющий всё за один шаг. Он
проверяет инструментарий, собирает argo-core, если доступен Rust, и
задаёт разумные настройки по умолчанию. Если вы предпочитаете сделать всё
вручную, запустите интерактивный мастер настройки.»

ON SCREEN:
```
cd argo-brain
python3 -m argo_brain setup
```

NARRATION: «Мастер настройки шаг за шагом проведёт вас через базовую
конфигурацию.»

### Сцена 5 — Проверка командой doctor (2:40–3:25)

ON SCREEN:
```
python3 -m argo_brain doctor
```

NARRATION: «Теперь выполните команду doctor. Это встроенная диагностика
ARGO — она проверяет вашу установку и сообщает обо всём, что выглядит
неправильно. Если все строки зелёные, вы готовы.»

### Сцена 6 — Первый запуск (3:25–3:50)

ON SCREEN:
```
python3 -m argo_brain chat
```

NARRATION: «И чтобы убедиться, что всё работает, запустите чат. Он
использует локальную тестовую модель, поэтому ему не нужен ни ключ API,
ни интернет. Напишите сообщение — и ARGO ответит.»

### Сцена 7 — Заключение и призыв к действию (3:50–4:10)

ON SCREEN: логотип ARGO, адрес GitHub и надпись «Далее: ваш первый
разговор».

NARRATION: «Вот и всё — ARGO установлен. ARGO распространяется под
лицензией MIT с открытым исходным кодом и сейчас находится в стадии
альфа на пути к общему релизу, поэтому ваш отзыв действительно важен.
Поставьте нам звезду на GitHub и смотрите следующее видео, где мы
разберём ваш первый настоящий разговор и работу с CLI. Спасибо за
просмотр.»
