# ARGO Tutorial Video Scripts

Text scripts for the ARGO v3.0 launch tutorial videos. There are
**12 scripts** — 4 topics, each in 3 languages (English, Russian,
Uzbek in Latin script).

Each script contains: a title, target duration (~3-5 minutes), a
scene-by-scene breakdown with `ON SCREEN` (visuals / commands) and
`NARRATION` (spoken voice-over), and a closing call to action.

For the `-ru` and `-uz` scripts the narration is written in that
language; `ON SCREEN` command and code lines stay verbatim so they match
what the viewer types.

## The 12 scripts

| # | Topic | English | Russian | Uzbek |
|---|---|---|---|---|
| 1 | Install ARGO in 5 minutes | `01-install-en.md` | `01-install-ru.md` | `01-install-uz.md` |
| 2 | Your first conversation & the CLI | `02-first-chat-en.md` | `02-first-chat-ru.md` | `02-first-chat-uz.md` |
| 3 | Connecting a Telegram bot | `03-telegram-en.md` | `03-telegram-ru.md` | `03-telegram-uz.md` |
| 4 | Deploying with Docker Compose | `04-docker-en.md` | `04-docker-ru.md` | `04-docker-uz.md` |

## Recording notes

- Record the English set first; it is the reference for timing and
  on-screen actions.
- Russian and Uzbek versions reuse the exact same screen recordings —
  only the voice-over changes. The `ON SCREEN` blocks are identical
  across all three languages of a topic.
- Keep each video within its target duration; the scene timestamps are
  guidance, not hard cuts.
- Use a consistent dark terminal theme across the whole series.
- Tone: honest and factual. ARGO is alpha approaching GA — the closing
  call to action says so in every script. Do not add hype.
- Recommended publishing order: 01 → 02 → 03 → 04, as a playlist per
  language.
