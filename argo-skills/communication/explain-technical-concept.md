# Explain a Technical Concept

1. Gauge the audience's existing knowledge level.
2. Start with a one-sentence plain-language definition.
3. Use a concrete analogy before introducing precise terminology.
4. Give a small worked example, then state the caveats.
5. Check understanding by inviting a question.
