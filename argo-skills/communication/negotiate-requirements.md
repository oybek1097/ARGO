# Clarify Ambiguous Requirements

1. Restate the request in your own words and confirm it back.
2. List explicit assumptions you are making.
3. Ask targeted questions about edge cases and success criteria.
4. Identify the must-haves versus the nice-to-haves.
5. Write down the agreed scope before starting work.
