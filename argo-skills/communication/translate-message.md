# Translate a Message Faithfully

1. Identify the source and target languages and the register (formal/casual).
2. Translate meaning, not words — preserve intent and tone.
3. Localise dates, numbers, currency, and idioms appropriately.
4. Keep names, brand terms, and code untranslated.
5. Note any phrase that has no clean equivalent for the requester.
