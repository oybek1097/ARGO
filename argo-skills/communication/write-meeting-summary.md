# Summarise a Meeting

1. Capture the decisions made, separately from the discussion.
2. List action items with an owner and a due date for each.
3. Note open questions that still need resolution.
4. Keep it scannable — headings and bullets, not prose.
5. Send within a day while context is fresh; tag the owners.
