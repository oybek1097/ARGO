# Compare Options Objectively

1. Define the criteria that actually matter for this choice.
2. Weight the criteria by importance.
3. Score each option on each criterion with evidence.
4. Note deal-breakers separately from weighted scores.
5. Present the comparison and a recommendation with reasoning.
