# Write Clear Step-by-Step Instructions

1. State the goal and the prerequisites up front.
2. Use numbered, imperative steps — one action per step.
3. Show expected output or state after key steps.
4. Anticipate common mistakes and address them inline.
5. Test the instructions by following them literally.
