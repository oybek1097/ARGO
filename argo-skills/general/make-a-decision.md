# Make a Well-Reasoned Decision

1. State the decision and the criteria for a good outcome.
2. List the realistic options, including doing nothing.
3. Evaluate each option against the criteria.
4. Consider reversibility — bias toward action for cheap-to-undo choices.
5. Decide, record the reasoning, and set a date to review the outcome.
