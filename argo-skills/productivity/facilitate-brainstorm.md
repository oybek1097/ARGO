# Facilitate a Brainstorming Session

1. Frame the problem clearly and set a time box.
2. Generate quantity first — defer all judgement.
3. Build on others' ideas before critiquing.
4. Cluster ideas and dot-vote to shortlist.
5. Convert the top ideas into owned next steps.
