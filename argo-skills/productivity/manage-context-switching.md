# Reduce Context Switching

1. Batch similar small tasks into a single block.
2. Protect uninterrupted blocks for deep work.
3. Capture incoming requests to a list instead of acting immediately.
4. Disable non-urgent notifications during focus blocks.
5. Review the capture list at fixed times each day.
