# Run a Team Retrospective

1. Set a blameless tone — focus on systems, not people.
2. Gather what went well, what did not, and what puzzled the team.
3. Group themes and discuss the highest-impact ones.
4. Agree on a few concrete, owned action items.
5. Review the previous retro's actions for follow-through.
