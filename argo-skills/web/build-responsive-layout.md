# Build a Responsive Layout

1. Design mobile-first; start with the single-column layout.
2. Use flexible units and CSS Grid/Flexbox over fixed pixel widths.
3. Add breakpoints only where the content actually breaks.
4. Test on narrow, tablet, and wide viewports plus a zoomed view.
5. Verify tap targets and text remain legible at every size.
