# Improve On-Page SEO

1. Audit titles and meta descriptions — unique, descriptive, sized right.
2. Ensure one `h1` per page and a logical heading hierarchy.
3. Add structured data (schema.org) for the relevant content type.
4. Check crawlability: `robots.txt`, `sitemap.xml`, and canonical tags.
5. Confirm fast load and mobile-friendliness; both affect ranking.
