# Roll Out Multi-Factor Authentication

1. Choose factors: TOTP or hardware keys over SMS where possible.
2. Provide secure recovery codes and a documented reset path.
3. Enforce MFA for admin and privileged accounts first.
4. Communicate the rollout and provide enrolment guidance.
5. Monitor enrolment and follow up with stragglers.
