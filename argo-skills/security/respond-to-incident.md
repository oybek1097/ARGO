# Respond to a Security Incident

1. Contain first — isolate affected systems to stop the spread.
2. Preserve evidence: snapshots and logs before remediation.
3. Eradicate the root cause and remove attacker access.
4. Recover services and verify they are clean before reconnecting.
5. Write a blameless postmortem with concrete preventive actions.
